Knowing how to navigate to each page is difficult without knowing which buttons to click.
Here are the strings to add to the url to take you to each page.

Login page: "/" or "/Login"
Signup page: "/Signup"
User home page: "/UserLanding"
Create quiz page: "/CreateQuiz"
User join quiz page: "/JoinQuiz"
Guest join quiz page: "/GuestJoinQuiz"
Quiz PIN Display page: "/PINDisplay"
Quiz question from teacher's perspective: "/TeacherDisplayQuestion"
Quiz question from student's perspective: "/StudentDisplayQuestion"
Results page: "/DisplayResults"
